# Welcome to my blog!

This is my first post on this new blog. I decided to build my portfolio using [Astro](https://astro.build/) because of its incredible performance and developer experience.

## Why Astro?

Astro allows me to use my favorite frameworks like **Preact** when I need interactivity, but keeps the rest of the site static and fast.

- **Speed:** Only ships JavaScript when necessary.
- **Flexibility:** Use any UI framework.
- **DX:** Great documentation and tooling.
